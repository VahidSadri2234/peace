# Mahya configuration set

این پوشه شامل چهار فایل تنظیمات است که برای مجموعه «مهیا» آماده شده‌اند. هر فایل یک پروفایل مجزا دارد و می‌توانید آن‌ها را مستقیماً در مخزن GitHub خود (`https://github.com/VahidSadri2234/mahya`) آپلود کنید.

## فایل‌ها
- `pc`
- `v4`
- `v4_withoutgeo`
- `v6`

## نحوهٔ انتشار در GitHub
1. اگر هنوز مخزن را کلون نکرده‌اید:
   ```bash
   git clone git@github.com:VahidSadri2234/mahya.git
   cd mahya
   ```
2. پوشهٔ `mahya` را از این پروژه کپی کنید و محتوا را داخل مخزن قرار دهید (به‌طوری‌که فایل‌های فوق در ریشهٔ مخزن باشند).
3. تغییرات را کامیت و پوش کنید:
   ```bash
   git add .
   git commit -m "Add mahya configuration files"
   git push origin main
   ```

پس از این مراحل، فایل‌ها در آدرس GitHub بالا در دسترس خواهند بود.

## اجرای سریع روی ویندوز
1. یکی از فایل‌های بالا را بسته به نیازتان انتخاب کنید (مثلاً `pc`).
2. نام فایل را به `config.json` تغییر دهید و آن را در کنار فایل اجرایی برنامهٔ «sing-box» یا ابزار مشابهی که نصب کرده‌اید قرار دهید.
3. در PowerShell به همان پوشه بروید و دستور زیر را اجرا کنید:
   ```powershell
   .\sing-box.exe run -c config.json
   ```
   - اگر از یک رابط گرافیکی استفاده می‌کنید، معمولاً گزینهٔ «Import Config» یا «Open Config» وجود دارد؛ همان فایل را به برنامه معرفی کنید.
4. پس از اجرا، پراکسی سیستم فعال می‌شود و می‌توانید اتصال را تست کنید. برای توقف، پنجرهٔ اجرا را ببندید یا `Ctrl + C` بزنید.
